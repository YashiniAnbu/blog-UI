# 📰 Blog UI

A responsive blog web application built with **React** and **React Router DOM**, featuring category-based filtering, dynamic blog detail pages, and a mobile-friendly hamburger navigation.

---

## 🚀 Features

- 🗂️ Category-based blog filtering (All, Full Stack, Data Science, Cyber Security, AI)
- 📄 Dynamic blog detail page via `/blog/:id` route
- 📱 Fully responsive design with hamburger menu on mobile
- 🔝 Sticky header with scroll shadow effect
- 🎨 Clean and minimal UI with smooth hover animations
- 🔗 Client-side routing with React Router DOM
